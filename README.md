# Zivan

Zivan is a modern social community platform built for focused discussions, community discovery, and user-driven content. It features a dark purple-and-black interface, community-based feeds using the `z/` prefix, post creation, threaded comments, voting, reactions, search, notifications, user profiles, and community creation tools.

The project is a complete MVP built with Next.js, React, TypeScript, and Tailwind CSS. It uses localStorage persistence with seeded demo data, making it easy to run, explore, and extend without requiring a backend setup.

## Highlights

- Purple/black responsive social feed UI
- Communities such as `z/subzivan`, `z/technology`, `z/gaming`, and more
- Sign up, sign in, and local session persistence
- Create text, link, and image posts
- Vote, react, comment, and reply
- Join communities and view personalized feeds
- Search posts, communities, and users
- Community creation wizard with validation
- Basic community moderation settings
- Seeded demo users, posts, comments, and notifications
